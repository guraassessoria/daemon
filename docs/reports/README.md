# Relatorios

Esta pasta agrupa relatorios gerados ou usados durante o processamento dos livros.

- `audits/`: auditorias de lacunas, categorias e qualidade.
- `catalog/`: resumo da montagem do catalogo por areas.
- `certification/`: resultados das travas/certificacoes por categoria.
- `granular/`: passadas de extracao granular e refinamentos.
- `manual-review/`: leitura por lotes, coerencia de paginas e revisao manual.
